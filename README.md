## 📌 Project Overview

This project presents a comprehensive **Starbucks Sales Analysis Dashboard** created using **Microsoft Power BI**. The dashboard transforms raw sales data into meaningful business insights, helping stakeholders understand sales trends, product performance, and customer purchasing patterns.

The goal is to enable **data-driven decision making** by providing an interactive and visually appealing dashboard.

---

## ✨ Dashboard Highlights

📈 **Sales Trend Analysis**
- Track sales performance over time
- Identify seasonal and monthly trends
- Monitor revenue growth

💰 **Key Performance Indicators (KPIs)**
- Total Sales
- Total Orders
- Average Sales
- Business Performance Metrics

📊 **Product Performance**
- Best-selling products
- Product category comparison
- Revenue contribution by product

🏪 **Store Performance**
- Compare sales across store locations
- Identify high-performing outlets
- Evaluate regional performance

🎯 **Interactive Filtering**
- Dynamic slicers
- Drill-down analysis
- User-friendly dashboard navigation
